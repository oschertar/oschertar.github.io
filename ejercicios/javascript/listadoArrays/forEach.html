1.Indica los tres argumentos del método forEach a un array. Demuestra su uso mediante un ejemplo.

Los tres argumentos del forEach son:

  1. El valor del elemento
  2. El índice del elemento
  3. El array que está siendo recorrido
Su uso sería el siguiente:
  1. Creamos un array con algunos elementos: array = [0,1,2,3,4];
  2. Usamos el forEach de la siguiente forma:
    array.forEach(valorElemento, indice, array){
    console.log("Array[" + indice + "] = " + valorElemento);
