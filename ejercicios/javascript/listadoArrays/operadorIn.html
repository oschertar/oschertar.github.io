2. Indica la utilidad del operador in con los arrays. Demuestra su uso mediante un ejemplo.

El operador in devuelve true o false si la propiedad que estamos especificando está o no en el array. Cuando hablamos de índices podemos comprobar si un índice está en el array o no. Si hablamos de arrays con propiedades, podemos comprobar si dicho array posee esa propiedad.

Ejemplo:
let animales = ["perro", "gato", "ardilla", "tortuga"];
0 in animales // devuelve true
3 in animales // devuelve true
6 in animales // devuelve false

"length" in animales // devuelve true (es una propiedad que los Arrays poseen)

let movil = {marca: "Xiaomi", modelo: "Note 3 Pro", camara: "Samsung", añoFabricacion: 2016};
"marca" in movil // devuelve true
"modelo" in movil // devuelve true
"tecnologia" in movil // devuelve false
