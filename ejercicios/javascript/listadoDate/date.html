1. ¿Cuántos constructores tiene el objeto predefinido Date()? Explícalos mediante ejemplos.

El objeto predefinido Date posee 4 constructores:

  • let d = new Date(); - Muestra la fecha actual con el horario GMT +0 (en España es una hora más que las que nos devuelve)
  • let d = new Date(milliseconds); - Crea una fecha según los milisegundos pasados por parámetros. Esta cantidad de milisegundos se le suma a una fecha inicial (año 1970 apróximadamente) y se calcula la nueva fecha para mostrarnos una actual. En nuestro caso hemos hecho: let dateMs = new Date(1478798609000);
  • let d = new Date(dateString); - Crea una fecha según la cadena dada. En dicha cadena se especifica primeramente el mes(en inglés), seguido del día, año, horas, minutos y segundos. En nuestro caso es: let dateString = new Date("November 10, 2016 18:24:00");
  • let d = new Date(year, month, day, hours, minutes, seconds, milliseconds); - En este último caso, le pasamos los argumentos uno a uno para crear una fecha con dichos valores. Ejemplo: dateCompleto = new Date(2016, 10, 10, 18, 28, 0, 0);
